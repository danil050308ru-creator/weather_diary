## Описание программы
*"Weather Diary — это консольное приложение для ведения дневника погоды. Позволяет
добавлять записи о температуре, осадках, фильтровать их по дате и строить графики
температуры."*
---
## Требования для запуска
Что нужно установить на компьютер:
- Python 3.10 или выше
- Библиотеки: `pip install matplotlib requests`
